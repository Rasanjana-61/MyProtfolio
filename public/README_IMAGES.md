# Portfolio Images Guide

Please add the following images to this `public` folder:

## Required Images:

### Home Section
- `home.jpg` - Background image for the home/hero section

### About Section
- `about.jpg` - Your profile/about photo

### Portfolio Section
Based on your updated portfolio items, you need:
- `portfolio1.jpg` - Online food recipe management system screenshot
- `portfolio2.jpg` - Car rental management system screenshot
- `portfolio3.jpg` - Online garment management system screenshot
- `portfolio4.jpg` - CrownCrest Hotel Reservation System screenshot

## Image Specifications:

### Home Background (`home.jpg`)
- Recommended size: 1920x1080px or higher
- Format: JPG
- Should be a professional background image

### About Image (`about.jpg`)
- Recommended size: 400x400px (square)
- Format: JPG
- Your professional photo

### Portfolio Images (`portfolio1.jpg` to `portfolio4.jpg`)
- Recommended size: 800x600px or similar aspect ratio
- Format: JPG
- Screenshots or mockups of your projects

## How to Add Images:

1. Place all images directly in this `public` folder
2. Make sure the file names match exactly (case-sensitive)
3. The images will be automatically loaded by the application
4. Refresh your browser to see the images
